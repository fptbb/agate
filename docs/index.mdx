Agate is a custom atomic desktop image built on top of [Fedora Bazzite](https://bazzite.gg/) using [BlueBuild](https://blue-build.org/). It ships as a signed container image you can rebase onto any compatible Fedora Atomic system in minutes, giving you a curated, reproducible desktop with development tools, gaming support, KDE theming, and security utilities pre-configured.

## Getting started

<Steps>
  <Step title="Check prerequisites">
    You need a running Fedora Atomic, Silverblue, or Bazzite system with `rpm-ostree` available.
  </Step>
  <Step title="Rebase to Agate">
    Run the rebase command to switch your system to the Agate image.
    ```bash
    sudo rpm-ostree rebase ostree-unverified-registry:quay.io/fptbb/agate:latest
    ```
  </Step>
  <Step title="Reboot">
    After the rebase completes, reboot to boot into your new Agate system.
    ```bash
    systemctl reboot
    ```
  </Step>
  <Step title="Verify the image (optional)">
    Confirm the image signature using cosign.
    ```bash
    cosign verify --key https://os.fpt.icu/cosign.pub quay.io/fptbb/agate
    ```
  </Step>
</Steps>
